# Online Marketplace with Django

Este repositório contém um projeto de mercado online desenvolvido com Django, um framework web em Python. Aqui estão as instruções para configurar o ambiente de desenvolvimento:

## Instruções

1. **Clonar o Repositório:**

    ```bash
    git clone git@github.com:igopereira1/django-online-marketplace.git
    cd marketplace
    ```

2. **Criação do Ambiente e Instalação de Dependências:**

    2.1. **Criação do Ambiente Virtual:**

    ```bash
    python3 -m venv .venv
    ```

    2.2. **Ativação do Ambiente Virtual:**

    ```bash
    source .venv/bin/activate
    ```

    2.3. **Instalação de Dependências:**

    ```bash
    pip install -r requirements.txt
    ```

3. **Teste do Servidor:**

    ```bash
    python manage.py runserver
    ```
